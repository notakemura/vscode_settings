### Sidebar
- show/hide
````
ctrl + b
````
- focus
````
ctrl + shift + e or ctrl + 0
````
### Terminal
- show/hide
````
ctrl + j
````
- open new terminal
````
ctl + shift + `
````
- return to editor
````
ctrl + 1
````
